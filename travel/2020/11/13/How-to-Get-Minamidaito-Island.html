南大東島へは那覇から「船」か「飛行機」で行くことが出来ます。
飛行機であればJALグループのRAC（琉球エアコミューター）の飛行機が出ているので迷うことは少ないと思いますが、船の場合は手続きに少し迷ったので、実際に利用した体験を元に情報をまとめてみました。
（当記事に掲載の運賃等は2020年11月当時のものです。）

船と飛行機の比較

まずは船と飛行機の比較を簡単に紹介します。

料金

飛行機がかなり割高で、片道で4倍以上、往復で3倍の差になります。
| | 大人片道 | 大人往復 | | —- | —- | —- | | 船 | 5,790円 | 11,010円 | | 飛行機 | 26,220円 | 33,540円 |

所要時間

船を利用する場合は船内で一泊することになります。
| | 所要時間 | | —- | —- | | 船 | 13~15時間 | | 飛行機 | 1時間 |

運航本数

船は本数が少ない上に悪天候による出港日の変更が多いので、飛行機が圧倒的に利用しやすいです。
ただし那覇発の場合、金~月の1便は7時45分発と早い時間になります。
| | 運航本数 | | —- | —- | | 船 | 不定期かつ週に1~2便 | | 飛行機 | 火~木は1日に2便、金~月は1日に1便 |

船を利用する場合

南大東島への船は大東海運株式会社が運航する定期船「だいとう」があります。
南台東島では、乗客がコンテナに入りクレーンで吊るされて上下船します。
他では出来ない貴重な体験なので、これを目当てで船を利用したい人もいるのではないでしょうか？
かく言う私もその一人でした。

運航スケジュール

定期船とは言っても出港日は週に1~2回で毎月バラバラなので詳しくは公式の運航スケジュールを確認して下さい。
また、悪天候による出港日変更が多いので注意するとともに、飛行機利用の可能性も考えておくなど、副案を用意しておくことをお勧めします。

定期船「だいとう」は那覇を出港後、南大東島と北大東島の2島へ寄港するのですが、どちらに先に寄港するかはその都度変わります。
運航スケジュールに「南」と書いてあれば「南大東島先航」、「北」と書いてあれば「北大東島先航」の時刻表になり入港時間が変わるので注意が必要です。

予約方法

予約はお電話または現地窓口で受け付けています。
乗船する場所ごとに予約窓口が違うのでご乗船までの流れで確認して下さい。

運賃

2020年11月時点では「大人片道 5,790円」「大人往復 11,010円」でした。
小人や学割運賃もあるので、詳しくは公式の旅客運賃表を確認して下さい。
運賃の支払いは現地窓口にて現金のみになります。
支払い出来るのは前日か当日のみで、時間も決まっているので注意してくだい。

出港が延期になった場合

私が乗る予定だった船が出港延期になったことがあるのですが、当日の8時過ぎに電話で連絡が来ました。
出港予定日を伝えられ、キャンセルするかどうかの確認がありました。

メリット

  • 飛行機に比べて安い
  • コンテナに乗って上下船する体験ができる
  • 一泊分宿泊費が浮く

デメリット

  • 出港が不定期で変更も多いため旅程を組み辛い
  • 揺れやすいので船酔いの恐れがある
  • 時間がかかる

飛行機を利用する場合

冒頭でも触れたようにJALグループのRACを利用することができます。
飛行機を利用したことがあれば基本的には同じなのですが、小型のプロペラ機なので機内持ち込み手荷物サイズが合計100cm以内（45cm×35cm×20cm以内）になります。
また、悪天候で船が出ない場合は座席が早く埋まる場合もあるので、利用する場合は早めの予約をお勧めします。

予約や時間の確認はJAL公式ホームページで簡単に出来るので、ここでの紹介は割愛します。

メリット

  • 船に比べて旅程を立てやすい
  • 1時間で行ける
  • 上空から島の様子を見ることができる（夏から12月ごろまでは青々と繁るサトウキビ畑を見下ろせます）

デメリット

  • 料金が高額
  • 金~月の飛行機は出発時間が早いものしかない

あとがき

いかがだったでしょうか？ 本土から遠く離れた離島ですが、飛行機は毎日運航していますし、意外にアクセスは容易です。
船を利用するとなると、今の時代には珍しくWebでの予約や支払いが出来ないので少し戸惑いますが、電話をすればその後の流れなどは丁寧に教えてくれました。
